# Sprint 13 — Case Workspace & Work Management

Commands:
- `/caseworkspace` — recent cases
- `/caseworkspace CASE_NUMBER` — search/open a case
- `/workboard` — office pending work board
- `/myworks` — signed-in staff member's assigned works

Features:
- unified case summary with hearing, work, timeline, document and fee signals
- pending/completed/overdue work counters
- case-linked work lists
- staff work queue
- one-tap local work completion
- automatic case timeline entry when a work is completed
- no separate Task record is created
